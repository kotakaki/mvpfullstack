# Portfolio Backlog Management Workflow

1. Thu thập backlog item mới từ người dùng, log hoạt động, hoặc summary.
2. Xác định team sở hữu chính.
3. Xác định agent sở hữu hoặc nhóm agent tương lai.
4. Gắn backlog item vào nhóm `Hiện có` hoặc `Kế hoạch tương lai`.
5. Gắn trạng thái và mức ưu tiên.
6. Lưu backlog theo đúng team.
7. Cập nhật agent portfolio khi có agent mới hoặc kế hoạch agent mới.
