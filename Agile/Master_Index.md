# Master Index - Agile Team

## Tổng quan

`Agile Team` gồm 4 agent:

1. `Agent 1`: giám sát hoạt động trong toàn bộ workspace
2. `Agent 2`: thư ký tóm tắt từng hoạt động và lưu log
3. `Agent 3`: tạo daily summary và weekly summary
4. `Agent 4`: quản lý backlog và agent portfolio
