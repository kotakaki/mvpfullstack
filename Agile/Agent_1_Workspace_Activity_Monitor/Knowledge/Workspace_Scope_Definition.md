# Workspace Scope Definition

## Phạm vi mặc định

Agent 1 theo dõi các thư mục cùng cấp với `Agile` trong workspace hiện tại.

Ví dụ:
- `IdeaTeam`
- `TrendTeam`
- `SaleTeam`

## Đơn vị theo dõi

- Thư mục workspace
- Run mới trong `Output`
- File markdown mới
- Script mới
- Commit mới

## Sự kiện ưu tiên cao

- Tạo team mới
- Tạo agent mới
- Tạo output mới
- Chạy test và lưu kết quả test
- Push lên remote
- Gửi tổng kết lên Google Chat
