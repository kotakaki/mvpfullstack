# Workspace Monitoring Workflow

1. Xác định phạm vi quét: các thư mục cùng cấp với `Agile`.
2. Thu thập tín hiệu thay đổi từ file, thư mục, output, và git.
3. Lọc ra các sự kiện có giá trị theo dõi.
4. Chuyển danh sách sự kiện cho Agent 2 để tóm tắt và lưu log.
5. Đánh dấu những sự kiện cần đưa vào daily summary / weekly summary cho Agent 3.
