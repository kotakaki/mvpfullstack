# Skill: UX Audit & Optimization

Kỹ năng rà soát và tối ưu hóa trải nghiệm người dùng dựa trên dữ liệu và tiêu chuẩn ngành.

## 1. UX Audit (Kiểm soát chất lượng trải nghiệm)
- **Heuristic Evaluation**: Đánh giá giao diện dựa trên 10 nguyên tắc của Jakob Nielsen.
- **Accessibility Audit (WCAG)**: Đảm bảo sản phẩm có thể sử dụng bởi tất cả mọi người (kể cả người khiếm khuyết).
- **Critical Path Analysis**: Phân tích các bước then chốt người dùng phải đi qua và loại bỏ rào cản (friction).

## 2. Optimization Techniques
- **A/B Testing Visualization**: Thiết kế các biến thể (Variants) trên Stitch để kiểm thử giả thuyết.
- **Micro-copy Optimization**: Tinh chỉnh ngôn từ trên giao diện (Buttons, Tooltips) để tăng tỷ lệ chuyển đổi.
- **Performance Design**: Tối ưu hóa UI để đảm bảo tốc độ tải trang nhanh nhất.

## 3. Feedback Implementation
- Kỹ năng tiếp nhận phản hồi từ Agent 3 (QA) và Agent 1 (Strategy) để điều chỉnh thiết kế theo hướng thực dụng nhất.
