# Output Guide

Mỗi nghiên cứu trend nên có một thư mục riêng trong `Output/`.

Ví dụ:
- `Output/2026-03-weekly-ai-trends/`
- `Output/social-commerce-signals-q2/`

Bên trong nên có `Master_Index.md` và các file output được prefix theo `Recommended Reading Order`.

Convention khuyến nghị:
- `01_Signal_Log.md`
- `02_MVP_Proposals.md`
- `03_Marketing_Trend_Insights.md`
- `04_MVP_Marketing_Angles.md`
- `05_Executive_Summary.md`
- `06_Trend_Report.md`
- `07_NotebookLM_Slide_Brief.md`

Nguyên tắc:
- Prefix số để người đọc biết ngay thứ tự đọc tài liệu.
- `Master_Index.md` giữ vai trò entry file, không bắt buộc prefix số.
- Nếu một research run không cần đủ mọi file, vẫn nên prefix theo thứ tự logic thực tế của run đó.
