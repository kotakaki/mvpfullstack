# Workflow: Flow 1 - Signal to Marketing Trends

1. Nhận `Signal_Log.md` từ Agent 1
2. Nhóm signal theo:
   - message shift
   - channel shift
   - content format shift
   - audience language
   - attention / saturation risk
3. Chọn 3-5 trend marketing đáng chú ý nhất
4. Viết `Marketing_Trend_Insights.md`
5. Không ràng output vào một MVP cụ thể
