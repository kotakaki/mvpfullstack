# 日経クロステック「ベトナムIT特集」出稿計画案（2026年4月版）

## 1. 施策の背景と目的
日本市場におけるIT人材不足の長期化と、既存システムのモダナイゼーション（レガシー刷新）需要の高まりにより、信頼できるオフショアパートナーへの期待が非常に高まっています。
本施策は、日本最大級の技術メディア「日経クロステック」の権威性を活用し、弊社のプレゼンスを「単なる開発会社」から「戦略的DXパートナー」へと引き上げ、質の高い商談機会を創出することを目的とします。

## 2. 戦略的ターゲット：日経クロステック「ベトナムIT特集」
*   **媒体の特性：** IT/製造業の意思決定層が中心の会員基盤（約230万人）。
*   **特集の意義：** 「ベトナムIT産業の底力」特設ページに参画することで、FPTやRikkeisoft等の主要プレイヤーと並ぶ「業界の主要プレイヤー」としての認知を即座に獲得できます。
*   **信頼の醸成：** 日経ブランドの文脈で語られることで、顧客の社内稟議が通りやすくなる「伴走効果」が得られます。

## 3. 具体的な掲載プラン案（ウェビナー資産の活用）
先日の**ウェビナーで提示した「技術ビジョン」と「成功事例」を再利用**し、効率的に高品質な記事を展開します。

| 項目 | 詳細内容 |
| :--- | :--- |
| **推奨プラン** | **Active Special（リードジェネレーション型）** |
| **投資コスト** | 2,500,000円（税別） |
| **獲得目標** | **約200件の高品質なリード** |
| **施策内容** | ウェビナー内容を基に日経のライターが記事を作成。記事末尾でウェビナー資料を「限定ホワイトペーパー」として提供し、顧客情報を回収。 |
| **活用素材** | ウェビナーでの「CEOビジョン」「生成AI活用トレンド」「レガシー刷新事例」 |

## 4. 本施策の3つのメリット
1.  **確実なリード獲得：** 日経のActive会員から、役職者やDX担当者のリードを効率的に獲得。
2.  **既存資産の最大活用：** ウェビナー素材をベースにするため、準備工数を大幅に削減。
3.  **営業・広報資産化：** 「日経クロステック掲載記事」実績は、商談時の信頼担保や採用ブランディングに強力な武器となります。
