# Quy trình: Phân tích website doanh nghiệp

1. Nhận domain hoặc website doanh nghiệp cần nghiên cứu.
2. Tìm sitemap và rà cấu trúc site trước.
3. Chọn các trang quan trọng theo `Company_Website_Signal_Framework.md` và kết quả từ sitemap.
4. Trích xuất thông tin quan sát từ từng trang ưu tiên.
5. Rút ra thông điệp, dấu hiệu ICP, dấu hiệu GTM và dấu hiệu tăng trưởng.
6. Ghi rõ nguồn trang cho mỗi nhận định quan trọng.
7. Tạo `03_Company_Website_Intelligence.md`.
