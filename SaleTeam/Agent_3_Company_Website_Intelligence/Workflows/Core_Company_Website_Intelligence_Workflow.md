# Quy trình: Phân tích website doanh nghiệp

1. Nhận domain hoặc website doanh nghiệp cần nghiên cứu.
2. Chọn các trang quan trọng theo `Company_Website_Signal_Framework.md`.
3. Trích xuất thông tin quan sát từ từng trang.
4. Rút ra thông điệp, dấu hiệu ICP, dấu hiệu GTM và dấu hiệu tăng trưởng.
5. Ghi rõ nguồn trang cho mỗi nhận định quan trọng.
6. Tạo `03_Company_Website_Intelligence.md`.
