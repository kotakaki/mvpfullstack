# mvpfullstack ワークスペース

複数の専門エージェントチームが連携し、アイデアの具体化からプロダクト開発、市場調査、セールス支援までを包括的に行うフルスタックMVP開発環境です。

## 構成チーム一覧

### 1. [IdeaTeam](./IdeaTeam/IdeaMVP_guideline_JP.md)
アイデアの具体化、MVPのドキュメント化（BRD/SRS/PRD）、およびUI/UX設計を担当します。5つの専門エージェントが連携し、生のアイデアを設計図へと落とし込みます。

### 2. [TrendTeam](./TrendTeam/README_JP.md)
市場のシグナルを調査・分析し、プロダクトやマーケティングに活用できるインサイトを提供します。トレンドに基づいたMVPの提案や競合調査を専門としています。

### 3. [SaleTeam](./SaleTeam/README_JP.md)
セールスワークフローを支援するために、リードの収集、アカウント（企業）調査、ソーシャルメディアの公開情報分析を行います。すぐに営業で使えるデータの提供を目標としています。

### 4. [Agile](./Agile/README_JP.md)
ワークスペース全体の活動を監視し、日報や週報の作成、バックログの管理、各チームのポートフォリオ管理を行う内部運用チームです。

### 5. [ContentTeam](./ContentTeam/README_JP.md)
AI検索時代（SGE/Perplexity等）に最適化された、2026年最新のSEOコンテンツ制作を担当します。実体験や一次情報を重視し、AI検索のソースとして選ばれやすい高品質なコンテンツを生成します。

## 日本語化について
各ディレクトリ内に `_JP.md` という名称の日本語版ドキュメントを配置しています。
主なガイドラインやREADMEは日本語で確認可能です。
