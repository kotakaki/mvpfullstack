# Skill: Feature Inventory Synthesis

Kỹ thuật tổng hợp và phân loại danh sách tính năng (Functional List) từ dữ liệu quét được.

## 1. Phương pháp phân tích
Lọc ra các động từ và danh từ kỹ thuật xuất hiện dày đặc trong trang Feature và Pricing.

## 2. Phân loại tính năng (Categorization)
Phân loại vào các nhóm để dễ so sánh:
- **Core Functionality**: Các tính năng bắt buộc phải có để giải quyết vấn đề chính.
- **Workflow Automation**: Các tính năng giúp tối ưu hóa thời gian (vd: auto-sync, bulk action).
- **Management & Admin**: Cài đặt, phân quyền, bảo mật.
- **Integration**: Kết nối với bên thứ ba (Slack, Zapier, v.v.).

## 3. Đánh giá mức độ quan trọng
- **Must-have**: Được quảng bá mạnh nhất trên Hero section.
- **Nice-to-have**: Nằm sâu trong bảng so sánh tính năng (Pricing table).

## 4. Output
Một bảng danh sách tính năng có mô tả ngắn gọn và phân nhóm rõ ràng, sẵn sàng để Agent 1 đưa vào Feature List của BRD.
