# Knowledge: Project Cataloging & Packaging

Mục tiêu: Tổ chức hồ sơ dự án khoa học, giúp bàn giao và tra cứu hiệu quả.

## 1. Folder Structure Standard
Mọi dự án MVP nên có cấu trúc đầu ra (`Output/`) như sau:
- `BRD/`: Tài liệu yêu cầu kinh doanh.
- `Design/`: Tài liệu thiết kế kỹ thuật (SDD).
- `Assets/`: Hình ảnh, sơ đồ, prototype.
- `Master_Project_Index.md`: Tệp điều hướng chính.

## 2. Naming Convention
- Sử dụng CamelCase hoặc Snake_case nhất quán cho thư mục.
- Tên tệp phải phản ánh nội dung và phiên bản (nếu cần).

## 3. Handover Package
Một gói bàn giao (Handover Package) tiêu chuẩn bao gồm:
- Toàn bộ hồ sơ KWSR của các Agent đã tham gia.
- Bản báo cáo tóm tắt (Executive Summary) cho cấp quản lý.
- Hướng dẫn triển khai nhanh (Quick Start Guide).
