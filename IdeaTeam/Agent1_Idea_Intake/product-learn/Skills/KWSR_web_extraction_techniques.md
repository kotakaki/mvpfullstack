# Skill: KWSR Web Extraction Techniques

Các kỹ thuật chuyên sâu để trích xuất tri thức từ nội dung web một cách hiệu quả.

## 1. Nhận diện Semantic Content
- Tập trung vào các thẻ `<h1>` đến `<h3>` để hiểu cấu trúc thông điệp.
- Phân tích nội dung trong các thẻ `<li>` hoặc thẻ chứa icon để tìm danh sách tính năng.
- Chú ý đến các đoạn text trong nút (CTA) để hiểu hành động mục tiêu.

## 2. Lọc nhiễu (Noise Reduction)
- Bỏ qua các phần Footer, Header điều hướng chung, và các banner quảng cáo.
- Tập trung vào "Main Content" hoặc các section có ID/Class liên quan đến `product`, `features`, `services`.

## 3. Phân tích đối thủ (Competitive Analysis)
- Tìm kiếm các bảng so sánh (Comparison tables) để thấy sự khác biệt.
- Đọc các Testimonials để hiểu nỗi đau (Pain points) của người dùng mà sản phẩm đang giải quyết.

## 4. Xử lý dữ liệu thô
- Chuyển đổi các mô tả dài dòng thành các "Idea Bullets" súc tích.
- Ghi chú các thuật ngữ (Terminology) đặc trưng mà website đó sử dụng.
