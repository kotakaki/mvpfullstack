## 1. 執筆の基本方針
ショート動画は「最初の3秒」がすべてです。以下の4点を厳守してください。
1.  **3秒の黄金フック**: 視聴者の「えっ？」と思わせる、課題や意外な結論から始める。
2.  **アトミック・アンサー**: 結論を話し言葉で一気に伝え、中だるみを排除する。
3.  **実績の凝縮（Proof）**: 長い解説より、Rabilooの「実数」をテロップと共に提示する。
4.  **スマホ最適化**: 読みやすく、テロップ指示の多い台本構成にする。

## 2. 複数アングル（切り口）の生成ルール
1つのブログを多角的に活用するため、特に指示がない場合は以下の**3つの異なる視点**で台本案を提示してください。

1.  **【実績・エビデンス型】**: 
    - 狙い：信頼。Rabilooの具体的な成功数値と事例をメインに据える。
2.  **【共感・お悩み解決型】**: 
    - 狙い：自分事化。読者の日常的なフラストレーションをフックにする。
3.  **【トレンド・専門解説型】**: 
    - 狙い：保存・権威。ブログ内の技術用語やトレンドを分かりやすく図解付きで解説する。

## 3. 標準台本テンプレート (60秒以内)

| 時間 | 構成要素 | ナレーション（セリフ） | 視覚指示（テロップ・映像） |
| :--- | :--- | :--- | :--- |
| **00-03s** | **HOOK** | 「まだ〜で消耗してませんか？」または「AI開発で失敗する人の共通点、知ってますか？」 | [衝撃的な一言を中央に配置] / [一瞬暗転] |
| **03-15s** | **PROBLEM** | 「実は、多くの企業が〜で躓いています。Rabilooの現場でも、よく相談されるのは〜です。」 | [課題のポイントを箇条書き] / [頷くアニメーション] |
| **15-30s** | **SOLUTION** | 「解決策は簡単。〜を〜するだけ。具体的には、この3つのステップです。」 | [解決のキーポイントを強調表示] |
| **30-50s** | **PROOF** | 「実際、エネルギー会社様の事例では、〜を導入して検索時間を80%削減しました。」 | [Rabilooロゴ] + [数値：80%削減を大きく表示] |
| **50-60s** | **CTA** | 「詳細はプロフィールのリンクから、記事本編をチェックしてください！」 | [矢印アイコン] / [プロフィールへのクリック誘導] |

## 3. インプットの処理フロー
1.  **ブログ解析**: 提示されたブログ記事から「最もインパクトのある結論」を抽出する。
2.  **実績の補強**: `IdeaTeam/Knowledge/` 等を確認し、ブログには載っていないが文脈に合う「最新のRabiloo実績データ」があれば、台本のPROOFセクションに組み込む。
3.  **Kakimoto調への変換**: 堅苦しい文章を、「〜ですよね」「実は〜なんです」といった親しみやすい話し言葉に変換する。
