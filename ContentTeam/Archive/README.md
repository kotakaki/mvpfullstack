# Archive（リサーチ資料倉庫）

> [!CAUTION]
> このフォルダ内のファイルは「保管専用」です。
> AIエージェントは、ユーザーから明示的に指定されない限り、
> このフォルダ内のファイルを参照してはなりません。

## 使い方

1. Gemini Deep Research や NotebookLM のレポートを、このフォルダにMarkdownで保存する
2. `/content-engine` 起動時に、使いたいファイルを明示的に指定する
3. 指定しなければ、AIはこのフォルダを一切参照しない

## 命名規則

`[ツール名]_[トピック]_[時期].md`

例：
- `gemini_offshore_cost_2026Q2.md`
- `notebooklm_hedspi_talent_2026.md`
- `competitor_analysis_enterprise_dx.md`
