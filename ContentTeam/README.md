# ContentTeam

This team is responsible for high-quality content production optimized for the AI search era (SGE, Perplexity, etc.).
It specializes in collecting primary information and generating structured content evaluated by AI search engines.

## Constituent Agents
1. **Agent_1_Signal_Topic_Researcher**: AI search trend and keyword analysis
2. **Agent_2_Experience_Compiler**: Primary info extraction and experience manifestation
3. **Agent_3_AISearch_SEO_Writer**: Answer Engine Optimization (AEO) writing
4. **Agent_4_Quality_Editor**: Quality verification and platform-specific conversion

## Workflow
1. Receive keywords/topics from TrendTeam etc.
2. Agent 1 analyzes AI search competitors and required elements.
3. Agent 2 constructs unique perspectives and real-life stories.
4. Agent 3 writes the article or content.
5. Agent 4 performs final review and outputs to `Output/`.
