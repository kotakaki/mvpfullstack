# ContentTeam (コンテンツチーム)

AI検索時代（SGE/Perplexity等）に最適化された高品質なコンテンツ制作を担当するチームです。
一次情報の収集と、AI検索エンジンに評価される構造化されたコンテンツ生成を専門とします。

## 構成エージェント
1. **Agent_1_Signal_Topic_Researcher**: AI検索のトレンドとキーワード分析
2. **Agent_2_Experience_Compiler**: 一次情報の抽出と体験の具体化
3. **Agent_3_AISearch_SEO_Writer**: アンサーエンジン最適化（AEO）執筆
4. **Agent_4_Quality_Editor**: 品質検証と各プラットフォーム形式への変換

## フォルダ構造
- `Agent_1_Signal_Topic_Researcher/`
- `Agent_2_Experience_Compiler/`
- `Agent_3_AISearch_SEO_Writer/`
- `Agent_4_Quality_Editor/`
- `Output/`: 最終成果物

## ワークフロー
1. `TrendTeam` 等からキーワードやトピックを受領。
2. Agent 1がAI検索での競合と求められる要素を分析。
3. Agent 2が独自の視点や体験談を構成。
4. Agent 3が記事またはコンテンツを執筆。
5. Agent 4が最終確認を行い `Output/` に出力。
