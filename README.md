# Rabiloo Unified Agent System (Rabiloo Brain)

本プロジェクトは、Rabilooの各専門チーム（Idea, Trend, Sale, Agile, Content）の知見を統合し、AIエージェントを活用してビジネス成果を最大化するための共有基盤です。

## 📍 組織の構造とナビゲーション

### 1. 組織の基本原則 (The Rabiloo Way)
すべての活動の核となる指針です。
- [Global_Guideline_JP.md](./Global_Guideline_JP.md): 共通のトーン＆マナー・倫理規定。

### 2. 知見の地図 (Master Knowledge Map)
どこにどのような専門知識があるかを管理しています。エージェントはこの地図を元に他チームのデータを参照します。
- [Master_Knowledge_Registry_JP.md](./Master_Knowledge_Registry_JP.md): 全チームの知見インデックス。

### 3. 各専門チームのディレクトリ
- [ContentTeam/](./ContentTeam/): SEO/GEO執筆・記事診断チーム。
- [IdeaTeam/](./IdeaTeam/): 開発実績・製品戦略・UI/UX設計チーム。
- [TrendTeam/](./TrendTeam/): 市場動向・技術シグナル分析チーム。
- [SaleTeam/](./SaleTeam/): セールス・顧客インテリジェンスチーム。
- [Agile/](./Agile/): ワークスペース監視・ドキュメント要約・進行管理チーム。

---

## 🚀 使い方

### マーケティング・コンテンツ制作
マーケティング担当者向けに、AIエージェントを即座に活用するためのポータルを用意しています。
- [マーケティング向け利用ガイド](./ContentTeam/Marketing_Team_Portal_JP.md)

### エージェントの実行
各エージェントのプロンプト（指示書）を使用する際は、必ずルートの `Global_Guideline_JP.md` と `Master_Knowledge_Registry_JP.md` を読み込ませることで、組織全体の最新知見に基づいた高度な回答が得られます。
